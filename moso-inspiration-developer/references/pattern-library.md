# Pattern Library

Use 2-5 patterns per inspiration. A pattern is useful only if it changes the next question, direction, or output.

Do not ask "what famous work is this like?" Ask "what generative structure is hidden here?"

## Literature Patterns

- Image anchor: one image carries the whole idea.
- Unfinished narrative: the power comes from what has not happened yet.
- Defamiliarization: ordinary things become strange.
- Absent center: something missing organizes the field.
- Repeated variation: a word, object, or gesture keeps returning differently.
- Interior monologue: the idea lives in private thought texture.
- Repressed desire: a blocked want creates pressure.
- Fate fracture: a small event opens a life split.

## Philosophy Patterns

- Concept reversal: reverse the ordinary definition.
- Binary tension: two values pull against each other.
- First-principles retreat: strip the idea back to its primitive question.
- Paradox exposure: the contradiction is the engine.
- Subject recovery: ask who is allowed to speak, choose, or see.
- Boundary of knowledge: the idea points at what cannot be known.
- Common sense failure: the ordinary explanation breaks.
- Question reversal: the real question is the opposite of the obvious one.

## Cinema Patterns

- Image first: a frame arrives before plot.
- Tension through omission: what is withheld creates pressure.
- Object destiny: a thing carries narrative fate.
- Suspended rhythm: delayed action matters more than action.
- Shot transformation: a camera move changes meaning.
- Sound trigger: a sound unlocks memory or conflict.
- Empty-space narrative: absence in the frame tells the story.
- Action exposure: a small action reveals the hidden state.

## Visual Patterns

- Single subject: one thing must dominate.
- Negative space: emptiness is active.
- Light-shadow opposition: lighting carries conflict.
- Material metaphor: surface texture expresses inner logic.
- Low-saturation emotion: restraint holds feeling.
- Living deformation: shape behaves like a creature or organism.
- Still-life narrative: objects imply a story without people.
- Asymmetric balance: imbalance creates life.

## Writing Patterns

- Opening hook: begin with pressure, not explanation.
- Subtext: what is not said carries the meaning.
- Cold treatment: emotion is stronger when underplayed.
- Anti-AI short sentence: cut generic smoothness with concrete phrasing.
- Object carries emotion: feeling enters through things.
- Delayed explanation: postpone the answer.
- Open ending: preserve unfinishedness.
- Repeated image: return to the same image with altered meaning.

## Selection Rules

- Pick fewer patterns when the input is fragile.
- Pick patterns from different domains when the user asks to ferment.
- Pick visual patterns first when the user asks for an image.
- Pick writing patterns first when the user asks for fiction or copy.
- Always name the risk: over-literary, too abstract, too literal, too commercial, too diagnostic, too pretty, too final.
