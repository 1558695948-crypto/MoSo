# Visual Catalyst

Use this file when the user asks to turn an inspiration into an image, visual direction, poster, cover, UI mood, photography concept, or image-generation prompt.

## Goal

Create a visual direction that preserves the core spark. Do not chase "prettiness" alone.

A good visual output has:

- A clear visual core.
- Composition.
- Palette.
- Lighting.
- Material.
- Camera language.
- Style boundary.
- Avoid list.
- Prompt and negative prompt when useful.
- Iteration options.
- Critic notes.

## Modes

### Image Development

For dreams, fiction, emotional fragments, or abstract scenes.

Emphasize:

- Main subject.
- Spatial relation.
- Light and shadow.
- Material metaphor.
- Atmosphere.
- What must remain unresolved.

### Design Prototype

For UI, poster, brand, product, campaign.

Emphasize:

- Function of the image.
- Hierarchy.
- Typography direction if needed.
- Layout constraints.
- Visual system, not decorative clutter.

### Aesthetic Training

For photography, cover, style analysis, or taste calibration.

Emphasize:

- What the user seems drawn to.
- What to avoid.
- 2-3 controlled variations.

## Prompt Shape

Use this shape when producing an image prompt:

```text
Core subject: ...
Scene/composition: ...
Lighting: ...
Palette: ...
Material/texture: ...
Camera/lens: ...
Mood: ...
Style boundary: ...
Must preserve: ...
Avoid: ...
```

## Visual Critic

Reject or revise if:

- The output is over-neon.
- It becomes cheap cyberpunk.
- It has generic AI fantasy gloss.
- The composition is too full.
- The palette feels muddy.
- Typography feels cheap or random.
- The core spark is gone.
- It is only beautiful and has no inner pressure.
- It ignores known user aesthetic signals.

## Iteration Options

Offer 2-3 variation paths:

- More restrained.
- More surreal.
- More physical/material.
- More cinematic.
- More product-like.
- More intimate.

Do not offer too many.
